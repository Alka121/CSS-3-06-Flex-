CSS-FLEX

The flex box or flexible box model in css is a one-diamentional layout model that has flexible and efficient layouts with distributted spaces.

Flexible box layout Module ,makes it easier to design flexible responsive laout structure without using float and positioning.

Properties for parents (flex container)

1)display:flex;

2)flex-direction:row;

3)flex-wrap:wrap;

4)flex-flow:row-wrap;(flex-flow + flex-wrap + flex-direction)

5)justify-content:center;

6)align-item:center;

7)align-content:space-between;

8)column-gap:20px;

9)row-gap:20px;

10)gap:20px;

properties for the children(Flex-Item)

1)flex:1;0r flex-grow:1;

2)flex-shrink:3;

3)order:2;

4)align-self:center;
